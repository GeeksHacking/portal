# GeeksHackingPortal.OidcWebPlayground

A small Nuxt-based playground for experimenting with OpenID Connect and client-side authentication flows. See Nuxt docs: https://nuxt.com/docs

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Production

```bash
pnpm build
pnpm preview
```

Use this project to test OIDC clients, redirect URIs, and local auth flows.
